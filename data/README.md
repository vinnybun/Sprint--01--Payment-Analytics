The dataset used for this project can be gotten from [Kaggle Financial Fraud Detection Dataset](https://www.kaggle.com/datasets/sriharshaeedala/financial-fraud-detection-dataset/data).

It isn't stored in this repository because the dataset is more than the expected size of 25MB and hence can't be uploaded unto this repository.

After downloading the dataset; which is in csv format , it can be placed in your local notebook and loaded with the code on the notebook file. Please note theat the file was renamed for ease of location, you can rename it to "payflow_dataset.csv" to ensure a smooth reproduceable result.
